# ☕ Cafeteria - Landing Page Responsiva

Este projeto é uma **landing page completa e responsiva** para uma cafeteria fictícia, desenvolvida com **HTML5, CSS3 e JavaScript**.
A proposta do site é apresentar a marca, destacar produtos do menu, exibir avaliações de clientes e facilitar o contato com a empresa de forma moderna e interativa.

## 📌 Funcionalidades

* ✅ Layout moderno e responsivo
* ✅ Menu de navegação com rolagem suave entre seções
* ✅ Link ativo no menu conforme a seção visível na tela
* ✅ Cardápio com produtos e preços promocionais
* ✅ Carrinho de compras interativo
* ✅ Adicionar e remover itens do carrinho
* ✅ Cálculo automático do valor total
* ✅ Badge com quantidade de itens no carrinho
* ✅ Mensagem ao confirmar pedido
* ✅ Avaliações de clientes
* ✅ Mapa incorporado com localização
* ✅ Botão fixo do WhatsApp no rodapé

* **HTML5** → Estrutura da página
* **CSS3** → Estilização e responsividade

## 🛒 Carrinho de Compras

O projeto possui um carrinho funcional desenvolvido em JavaScript, com as seguintes ações:

* Adição de produtos
* Remoção individual de itens
* Contador visual no ícone do carrinho
* Total atualizado em tempo real
* Estado vazio quando não há produtos
* Finalização de pedido

## 🚀 Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/vitVasconcelos/cafeteria.git
```

2. Acesse a pasta:

```bash
cd cafeteria
```

3. Abra o arquivo `index.html` no navegador.
